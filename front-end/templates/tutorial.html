    A HackPack from DecisionCandy

    A quick and dirty package of Django on Google App Engine running Python 2.7 with HTML5 Boilerplate and Twitter Bootstrap.

    Use it to hack something out quickly without worrying about the standard drudgery of:

    • Setting up server stuff (other than a GAE account--two flicks of a whistle'll get you there).
    • Setting up all the basic Django configurations, including compatibility with GAE.
    • Ensuring everything's good to go with Python 2.7, a new feature (still experimental) on GAE.
    • Proper <!doctype>, <meta>, and all that stuff.
    • Consistent cross-browser styling and HTML5 pseudo-support for older browsers.
    • A basic level of decent styles, pretty buttons, and some JS goodies from Bootstrap.
