    Get Started

    This won't hurt a bit.

    1. Install Python 2.7

      If you haven't already done this, though, you'll probably have trouble writing Python logic.

      Assuming you're mostly on the front-end team, go here and do what it says. Note that if you're on a Mac, you probably already have Python. It may or may not be the correct version; be sure to check.

    2. Install the Google App Engine SDK

      No worries if you don't have experience with GAE. Just hit up this page and do what it tells you.

    3. Install Django

      I'm actually not sure this is necessary. I don't think it is. Let me double-check.

    4. Get Git (if not installed)

      I'm serious. You need to do this.

      Download and install Git here.

      Set up an account with either GitHub or BitBucket. GitHub's been "rocking git" for longer, but BitBucket offers free unlimited *private* repositories, with unlimited collaborators, if you have a .edu email address (aka if you go to college).

    5. Download this thing.

      Go here and click "fork" in the upper right to fork the repository. This will be your application, so give it the name you want, etc. Now might also be a good time to add your teammates as collaborators, but that could also be done later.

      Once you're ready to move on, open command prompt (or bash or whatever) and cd (change directory, for the unititated) to the parent directory of where you'd like to keep this. For example, I have a 'djcode' folder where I keep all my code; I would type cd c:/djcode .

      Clone the repository you forked (aka, the one you just 'made') from bitbucket or git. This should be something like git clone https://bitbucket.org/username/appname.git. You'll see the url towards the top of the repository page.

      You can get the full thing with docs (basically what you're looking at) here ('get source' or 'fork' in the upper right), but you should use it more for reference than as a template. Especially if you're a beginner, I would recommend taking a look at it anyway as it was built expressly with reuse and 'teaching' in mind. We'll get to this later.
